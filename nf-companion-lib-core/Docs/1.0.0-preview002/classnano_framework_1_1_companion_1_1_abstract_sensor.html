◆ HasSensorValueChanged()

abstract bool nanoFramework.Companion.AbstractSensor.HasSensorValueChanged ( )
pure virtual

The sensor driver implementation should decide what is the meaning of change of a sensor value. We just check if the value of sensor has changed or not. Some sensor driver implementors may not want this automatic check and may want to have a polling mechanism for the client applications. The client apps,can use this method in their own polling implementation and check for value changes

Returns
If true, then sensor value has changed else false
